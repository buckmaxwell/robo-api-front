Get Feed

request feed:

Endpoint: crimenut.maxwellbuck.com/reports/feed
requires:

  • json with valid token*
  • lon*
  • lat*
  • page
  • miles
returns: list of reports as a json (see ex 5) or error message.

page: This is for pagination. Pages are up to 20 reports in length. The first page fetched is “page”:1 (1 is default if page not supplied). If there are next and/or previous pages the response will include a key called next_page and/or prev_page.

miles: this value determines how many miles away from lon and lat that reports will be shown. for example, “miles”:”25” would show reports up to 25 miles away in the feed. Default is 10.

*api required fields.
